# Used Car Price Prediction & Analysis Dashboard

ðŸ”¹ Main Project Description

In order to forecast car prices based on important characteristics like brand, ownership, fuel type, transmission, engine capacity, kilometers driven, and manufacturing year, this project analyzes data from the used car market.

In order to provide insightful dashboards and an understanding of price prediction, the project integrates data analysis, visualization, and machine learning concepts. Used car pricing trends, patterns, and relationships are investigated using a Tableau interactive dashboard.

The dashboard enables users to dynamically filter data and comprehend how various factors affect used car prices.
